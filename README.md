# 🚀⚛️ REACT USER LIST APP ⚛️🚀

A simple and responsive React application that fetches user data from an API and displays it beautifully in card format.

---

## ✨ Features

🌐 Fetches live user data from API
📦 Displays users in clean card layout
⏳ Loading indicator while fetching data
📱 Fully responsive design (mobile + desktop)
🎯 Clean UI with hover effects
⚡ Fast and lightweight React app

---

## 🖼️ Preview

Each user card includes:

👤 Name
📧 Email
📞 Phone
🏙️ City
🏢 Company

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# 📥 Clone the repository
git clone https://github.com/your-username/react-user-list-app.git

# 📂 Move into project folder
cd react-user-list-app

# 📦 Install dependencies
npm install

# ▶️ Start development server
npm run dev
```
